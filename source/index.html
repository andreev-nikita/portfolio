Метроном

Метроном разработан на JavaScript с применением объектно-ориентированного подхода. Для воспроизведения звуков используется Web Audio API. Метроном позволяет менять темп воспроизведения, музыкальный размер, ритмические рисунки, громкость. Для расчета точного времени воспроизведения звуков используется встроенный в Web Audio API таймер, в обход таймера JavaScript и методов SetTimeOut и SetInterval.

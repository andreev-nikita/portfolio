Несмотря на то, что еще в детстве я увлекался программированием, настоящая страсть к разработке ПО появилась у меня лишь в процессе обучения в университете. В разработке программного обеспечения, я всегда уделяю время качеству выполнемых задач. В процессе работы стремлюсь получить качественый, читаемый, расширяемый код. Также я слежу за новыми тенденциями в разработке ПО, и применяю их в работе при необходимости, использую современные инструменты. JavaScript
Объектно-ориентированный подход, работа с DOM, ECMA Script 6, Babel, WEB audio API, AJAX

Портфолио

Метроном

Метроном разработан на JavaScript с применением объектно-ориентированного подхода. Для воспроизведения звуков используется Web Audio API. Метроном позволяет менять темп воспроизведения, музыкальный размер, ритмические рисунки, громкость. Для расчета точного времени воспроизведения звуков используется встроенный в Web Audio API таймер, в обход таймера JavaScript и методов SetTimeOut и SetInterval.
